## 投票可能な参加者と投票不可な参加者の表示方法

単に参加者一覧中投票可能を上に、投票不可を下にソートして字を暗くする
→一覧の順序は変えない。失格者は全体的に暗くする。

「投票可能」「失格」って銘打って、投票可能動画はその動画へのリンクを、失格動画は参加者名の横にでも失格理由を添える感じ
→失格者も動画リンクをはる。
